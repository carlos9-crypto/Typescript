¿Que es Typescript y para que sirve?

TypeScript es un lenguaje de programación creado y mantenido por Microsoft. Se define como un "superset" (superconjunto) de JavaScript, lo que significa que cualquier código escrito en JavaScript (ya sea ES5 o versiones modernas como ES6+) es código TypeScript válido. Su propósito principal es permitirnos escribir aplicaciones de manera más estructurada y sensata.

La principal diferencia: Tipado Estático

A diferencia de JavaScript, que es un lenguaje de tipado dinámico donde las variables pueden cambiar de tipo fácilmente (lo que suele causar errores), TypeScript introduce un sistema de tipos estáticos.

Detección de errores: TypeScript te avisa mientras escribes el código si estás haciendo algo mal, como intentar asignar un número a una variable que definiste como texto (string) o pasar argumentos incorrectos a una función. Robustez: Este sistema ayuda a crear software más robusto, mantenible y con menos errores.

Ventajas y herramientas clave

Soporte de IDE: Si usas editores como VS Code, el soporte es excelente. El editor te indicará inmediatamente si hay un error de tipo en tu código. Documentación automática: Los tipos actúan como una forma de documentación. Al revisar un proyecto nuevo, puedes ver qué argumentos recibe una función o si son opcionales con solo pasar el ratón sobre ellos, sin necesidad de leer toda la lógica interna. Inferencia de tipos: TypeScript es capaz de "deducir" el tipo de una variable por el valor que le asignas inicialmente. Si luego intentas cambiarlo a otro tipo incompatible, el compilador te detendrá. Tipado estructural: TypeScript utiliza "duck typing", lo que permite describir la "forma" de un objeto o función para asegurar que coincida con lo esperado por el compilador.

Funcionalidades avanzadas

TypeScript no solo ofrece tipos, sino que también da acceso inmediato a las últimas características de ECMAScript (la especificación oficial de JavaScript) y añade herramientas que JavaScript no posee de forma nativa, como interfaces, decoradores y tipos avanzados
